🚀 HOW TO USE:

  1. LOGIN TO ADMIN PANEL
     http://your-domain/admin/login

  2. UPDATE SEO SETTINGS
     • Click "SEO Settings" in sidebar
     • Update global SEO (title, description, OG image)
     • Update page-specific SEO for each page
     • Click "Save SEO Settings"

  3. UPDATE GENERAL SETTINGS
     • Click "Settings" in sidebar
     • Update site name, contact info, social media
     • Click "Save Settings"

  4. CLEAR CACHE (Important!)
     Run after making changes:
     
     cd /var/www/sjml
     rm -rf .next/cache
     pm2 restart sjml-app

  5. VERIFY CHANGES
     • Hard refresh browser (Ctrl+Shift+R)
     • View page source to see meta tags
     • Check with SEO tools

📊 DATABASE STRUCTURE:

  SEO Data: stored in "content" collection
  • pageKey: "seo"
  • Contains: globalTitle, globalDescription, pages array

  Settings Data: stored in "content" collection
  • pageKey: "settings"
  • Contains: siteName, contactEmail, socialMedia, etc.

═══════════════════════════════════════════════════════════════════

🔧 TECHNICAL DETAILS:

  SEO Implementation:
  • Uses Next.js generateMetadata() function
  • Fetches data server-side on each request
  • Falls back to global settings if page-specific not set
  • Supports Open Graph and Twitter Cards

  Settings Implementation:
  • Separate API endpoint from SEO
  • Stores general website configuration
  • Includes contact info and social media links
  • Can be extended with additional fields
